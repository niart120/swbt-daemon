# 初期仕様

`swbt-daemon` の初期方針と長期的に残す設計文脈を置く。

`tmp/` より長く残す安定した計画資料は、このディレクトリに置く。
作業中の実装作業は work unit record として `work-units/wip/local_{nnn}/` に置く。
完了した work unit record は `work-units/complete/local_{nnn}/` に移す。

現在の初期仕様:

- `SWBT_DAEMON_INITIAL_DIRECTION.md`
- `REPOSITORY_INITIALIZATION_TODO.md`
- `BTSTACK_SWITCH_DEVELOPMENT_PLAN.md`
- `BTSTACK_SWITCH_DAEMON_IPC_DESIGN.md`
